Portfolio

Class Sync

Live Engagement Monitoring Web App for Remote Classrooms. ClassSync is the next big tool in remote learning engagement monitoring for teachers. In light of the 2020 pandemic creating an atmosphere where learning has become more digital and remote, we understand that teachers need solutions for optimizing their students' engagement. As the learning environment continues to shift to a remote setting, we want to give teachers a tool to get back their ability to personalize their interactions with their students.

Sneaker Shopper

Sneaker shopper is an e-commerce website that specializes in the sale of athletic footwear. It is visually stunning and has a functioning cart built using browser local storage for guests and database for users.

No Bad Apples

A game built using Phaser.io library using sprites, collision, and audio. To win, try to only collect the good apples and avoid the rotten ones. It's easy to play because it's simply tracks your left and right movements so it's great for all ages!

Trivia

Trivia is a serverless game that parses through a json datalist to display multiple choice questions and answers. It ultilizes local storage and redux to rehydrate the current player's progress. This project was built using create-react-app, it allows a player to make one selection, submit it while seeing their score update in real time, and correct/incorrect choices are displayed in green and red font color respectively.
